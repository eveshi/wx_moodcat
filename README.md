## Mood Dog

### About

Mood Dog is a mini-app for people to write mood diaries. It is based on Wechat-WeApp.  

### Setup

1. Download Wechat DEV Tools.
2. 
```
git@github.com:eveshi/wx_mooddog.git
```
3. Import the project to Wechat DEV Tools.
